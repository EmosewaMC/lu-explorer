What is this?

This web-application is designed to be an exhaustive resource on the content of the latest published version of the now discontinued game LEGO® Universe. It is implemented using a REST API that serves data from a game client in web-compatible formats. This unique approach means that LU-EXPLORER is able to replicate the majority of the content that is available in the traditional wiki at legouniverse.fandom.com in a way that's much closer to the original game mechanics.

This project was created by @Xiphoseer and has a GitHub repository located at https://github.com/xiphoseer/lu-explorer. Feel free to report any issues or suggestions on the issue tracker.

Finally, none of this would have been possible without the amazing work by the folks at NetDevil and the LEGO Group, which are listed in the LU Credits!
