What is this?

This web-application is designed to be an exhaustive resource on the content of the latest published version of the now discontinued game LEGO Universe. It is implemented as a wiki-esque interface to LU-JSON, which is a machine-readable and preprocessed export of the games' database and files. The images that this app uses are taken from LU-RES, which mirrors the game resources but in modern / web-compatible formats.

This unique approach means that LU-EXPLORER is able to replicate much of the content that would otherwise only be available in the traditional wiki at legouniverse.fandom.com, and also provide the necessary technical detail and metadata to understand how the systems in the game worked.

This project was created by @Xiphoseer and has a GitHub repository located at https://github.com/xiphoseer/lu-explorer. Feel free to report any issues or suggestions there.

Finally, none of this would have been possible without the amazing work by the folks at NetDevil and the LEGO Group, which are listed in the LU Credits!
